Introduction

@PLUGIN_NAME@ is an open source plug-in for IBM Rational Developer for i. It formats free-form RPG IV (RPGLE) source code to improve readability and consistency.

@PLUGIN_NAME@ features are driven from our ideas and needs, but everybody is encouraged to contribute suggestions and manpower to improve the power of @PLUGIN_NAME@.

Features

  • Formats free-form RPG IV (RPGLE) source code.
  • Configurable keyword casing (upper case, lower case, capitalized).
  • Configurable indentation of nested code blocks.
  • Formats procedure and keyword parameters.
  • Format source members directly from the Remote Systems view.
  • Format source members directly in an Lpex editor view.
  • Optionally executes the IBM formatter.
  • Optionally format source on save.
